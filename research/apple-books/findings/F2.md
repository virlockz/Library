# Apple Books iPhone Reader Screen — Detailed Design Findings

Research compiled from Apple Support documentation, Apple's Human Interface Guidelines, and multiple tech publication analyses.

---

## 1. Page Layout — Margins, Padding, Content Area, Background Color

**Content Area:**
- **Single-column layout** on iPhone (double-column available on iPad via "Allow Multiple Columns" theme setting)
- Content fills most of the screen between top and bottom margins
- Left and right edges (~20% width each) are tap zones for navigation, not content area
- Content text sits slightly inset from screen edges

**Margins:**
- Default side margins: ~15–20pt padding on each side on standard iPhone display
- Adjustable via **Themes & Settings > Customize > Accessibility & Layout Options** slider
- "Both Margins Advance" toggle (Settings > Apps > Books) makes either margin tap advance to next page

**Background Colors (4 Page Color Themes):**

| Theme | Color | Hex Estimate |
|-------|-------|-------------|
| White | Pure white (default) | `#FFFFFF` |
| Sepia | Warm yellowish-tan | ~`#F5ECD7` |
| Gray | Medium gray | ~`#D1D1D6` |
| Dark/Black | Near-black with light text | `#000000` or `#1C1C1E` |

**Named Themes (iOS 17+):** Original, Quiet, Paper, Bold, Calm, Focus — each combines preset font, background color, and spacing. Long-press to customize.

**Auto-Night Theme:** "Automatic" mode switches Light/Dark based on time of day or system Dark Mode.

## 2. Typography — Default Font, Size, Line Height, Spacing, Headings

**Available Fonts (11 total):**

| Font | Type | Notes |
|------|------|-------|
| **Athelas** | Serif | Book-themed default for many Apple themes |
| **Original** | Publisher font | Uses embedded EPUB font |
| **New York** | Serif | Apple's system serif |
| **Charter** | Serif | Elegant, literary |
| **Georgia** | Serif | Classic |
| **Iowan** | Serif | Apple-designed for Books |
| **Palatino** | Serif | Humanist |
| **San Francisco** | Sans-serif | Apple's system font |
| **Avenir Next** | Sans-serif | Geometric |
| **Seravek** | Sans-serif | — |
| **Times New Roman** | Serif | Classic |

**Default Font:** Most Apple-designed themes default to **New York** or **Athelas** (serif) for literary feel.

**Font Size:**
- Adjustable via `aA` appearance controls
- No single fixed default — adapts to theme and book
- Apple HIG minimum: 11pt; reading fonts typically default to ~17–20pt

**Line Height / Spacing:**
- Default line spacing: ~1.4–1.5x font size (optimized for readability)
- Character spacing and word spacing independently adjustable via sliders

**Paragraph Spacing:**
- Determined by book's CSS/EPUB markup; Apple Books respects publisher's styles
- "Justify Text" toggle: ON by default (full-width lines; ragged right when off)

**Heading Style:** Rendered per book's markup and embedded styles — Apple Books does NOT override heading typography.

**Auto-Hyphenation:** Enabled by default (Settings > Books > Auto-Hyphenation).

## 3. Page Turn Animation — Transition Types, Timing, Easing

**Three Animation Options (since iOS 16.4):**

| Option | Description | Duration | Easing |
|--------|-------------|----------|--------|
| **Curl** (default) | Realistic 3D page-curl from bottom-right corner; mimics physical paper | ~0.5–0.7s | Spring-based (fast start, gentle deceleration) |
| **Slide** | Flat horizontal slide left/right | ~0.3–0.4s | Smooth ease-in-out |
| **None** | Instant swap, no transition | 0s | N/A |

**Access:** Tap screen → Menu button (bottom-right) → Themes & Settings → Page-turning icon (page with bent corner) → Choose Curl/Slide/None

**History:** Curl was the only option from original iBooks through iOS 15. Apple removed it in iOS 16.0 (replaced with Slide as default), then reinstated Curl in iOS 16.4 due to user demand.

## 4. Top Bar — Close Button, Title, Bookmark, Search, Animation

**Visibility:** Hidden by default. Appears only when user **taps the center of the screen**.

**Elements (left to right):**

| Position | Element | Description |
|----------|---------|-------------|
| Left | **Back/Close button** | Chevron `‹` or `X` to return to library |
| Left (secondary) | **Return arrow** | Rounded arrow to return to previous reading location |
| Center | **Book title / Chapter name** | Displayed in the center |
| Right | **Bookmark ribbon icon** | Toggle bookmark on current page |

**Animation:**
- Appears with **fade + slide-down** from above the viewport
- Auto-hides after a few seconds of inactivity or when user taps content area again
- Top bar and bottom toolbar appear/disappear together in coordinated animation

## 5. Bottom Toolbar — Visibility, Contents, Progress Bar

**Visibility:** Hidden by default during reading. Appears simultaneously with top bar on **center-screen tap**. Auto-hides after inactivity.

**Contents (left to right):**

| Element | Icon/Label | Function |
|---------|-----------|----------|
| **Search** | Magnifying glass | Opens book search interface |
| **Table of Contents** | List icon | Opens chapter/navigation list |
| **Font/Appearance** (`aA`) | `aA` icon | Quick access to font size, themes, appearance |
| **Brightness** | Horizontal slider | Adjusts screen brightness within Books (separate from system) |
| **Menu** | Two lines over three dots (bottom-right) | Opens full Themes & Settings panel |

**Progress Bar:**
- **Thin horizontal line** at the very bottom edge of the screen
- ~2–3pt tall, solid colored bar (accent color of current theme)
- Fills from left to right proportional to reading progress through entire book
- **Visible even when toolbar is hidden** — subtle persistent indicator
- **Draggable scrubber** — touch and drag to quickly navigate to any point in the book
- Tooltip/popover shows percentage or chapter location when scrubbing

## 6. Page Curl Effect — Realistic 3D Curl or Simple Slide?

**It is a realistic 3D curl.**

The Curl animation renders the page as if it were physical paper being lifted from the bottom-right corner:

- **3D perspective distortion** of the turning page
- **Shadow casting** underneath the lifted page corner
- **Gradient of light and shadow** on the curling surface (underside is slightly darker)
- **Interactive finger tracking** — curl follows user's finger position in real time during swipe
- **Physical snap-back** if user releases before page reaches halfway point
- **Reverse direction** (going back) curls from the left corner

This is distinctly different from a flat slide — it has depth, shading, and curved geometry.

## 7. Swipe Gestures — Page Turns, Dismiss

**Page Turn Gestures:**

| Gesture | Action |
|---------|--------|
| **Swipe right-to-left** | Advance to next page |
| **Swipe left-to-right** | Go back to previous page |

**Tap Zones (alternative to swiping):**

| Zone | Width | Action |
|------|-------|--------|
| Right margin | ~20% of screen | Advance to next page |
| Left margin | ~20% of screen | Go back to previous page |
| Center | ~60% of screen | Toggle top bar + bottom toolbar |

With "Both Margins Advance" enabled, both left and right taps advance the page; only swipe left-to-right goes back.

**Dismiss Gesture:**
- **Swipe down** (pull-down): **Closes the book** and returns to library
- This is a natural iOS modal dismissal pattern

**Other Gestures:**
- **Pinch to zoom**: Available for fixed-layout EPUBs (children's books, cookbooks) but NOT for standard reflowable EPUBs
- **Touch and hold on progress bar + drag**: Quick scrubbing through table of contents

## 8. Reading Progress — Display Methods

**Progress Bar:**
- Thin, narrow horizontal bar at bottom edge of screen (~2–3pt tall)
- Solid colored line (theme accent color) filling left-to-right
- Proportional to progress through **entire book** (not just current chapter)
- Visible during reading even when toolbar is hidden

**Percentage / Location:**
- Shows **percentage** (e.g., "42%") or **location number** when user interacts with progress scrubber
- Format varies: percentage, location number, or chapter name depending on context
- No persistent page numbers displayed during normal reading (unlike Kindle)

**Chapter-Level Progress:**
- Progress bar = entire book progress
- Chapter navigation handled through Table of Contents panel

**Reading Goals (iOS 13+):**
- Tracks daily reading time
- Shows "Reading Goals" notification when daily goal is met
- Home/library screen shows reading streak and total books completed
