## 1. Typography / Font Panel

### Access Path
Tap the **menu button** (icon: two horizontal lines + three dots) in the bottom-right corner → **Theme & Settings** → **Font Size / Options**

### Controls Present

| Control | Type | Details |
|---------|------|---------|
| **Font Size** | Slider with A/A buttons | Small "A" on left (decrease), large "A" on right (increase). Continuous slider between them. Range: approximately 12pt–40pt equivalent |
| **Font Selection** | Scrollable list | List of available typefaces displayed vertically. Default font is **San Francisco** (system font). Also includes **New York** (serif), **Georgia**, **Palatino**, **Athelas**, **Times New Roman**, and other system/installed fonts |
| **Bold Text** | Toggle switch | On/Off toggle to apply bold weight to current font |
| **Line Spacing** | Slider or stepped control | Adjusts leading between lines. Accessible via **Options** submenu |
| **Word Spacing** | Slider | Adjusts spacing between words |
| **Character Spacing** | Slider | Adjusts tracking between characters |
| **Full Justification** | Toggle switch | Toggles between left-aligned and fully justified text |

### Visual Layout
- The font panel slides up from the bottom as a **half-sheet** (bottom sheet)
- Font size controls are prominent at the top
- Font family list is a vertical scrolling list below
- Each font name is rendered in its own typeface as a preview
- Checkmark (✓) indicates currently selected font
- The panel has a **grab handle bar** at the top center for dismissal

### Typography in iOS 16+
- Font size uses **dynamic type** scaling
- Slider thumb is a white circle on a light gray track
- The "A" labels flanking the slider use SF Pro with different weights

## 2. Theme Panel (Background Themes)

### Access Path
Tap menu button → **Theme & Settings**

### Theme Options

The theme selector displays **background color swatches** in a horizontal row:

| Theme | Background Color | Text Color | Display Style |
|-------|-----------------|------------|---------------|
| **White** | `#FFFFFF` pure white | Black | Circular swatch |
| **Sepia / Cream** | `#F5E6C8` warm cream | Dark brown | Circular swatch |
| **Light Gray** | `#E8E8E8` light gray | Black | Circular swatch (new in iOS 16) |
| **Dark Gray** | `#333333` dark gray | White | Circular swatch |
| **Black (Dark Mode)** | `#000000` pure black | White | Circular swatch |

### Additional Theme Modes
- **Match Device Settings** — follows system Light/Dark mode
- **True Tone** — adapts based on ambient lighting conditions
- **Light Mode** — forces light backgrounds
- **Dark Mode** — forces dark backgrounds

### Visual Display
- Themes are shown as **filled circles** (approximately 36–44pt diameter)
- Arranged in a **horizontal row** at the top of the Theme & Settings panel
- Selected theme has a **ring/border highlight** around it
- Tapping a swatch immediately applies the background color
- The row is horizontally scrollable if more themes exist

### Color Palette (Approximate Hex Values)
```
White:     #FFFFFF  (pure white background)
Cream:     #F5E6C8  (warm sepia/parchment tone)
Gray:      #E8E8E8  (cool light gray, new iOS 16)
Dark Gray: #333333  (medium dark)
Black:     #000000  (AMOLED-friendly pure black)
```

## 3. Table of Contents Panel

### Access Path
Tap menu button → **Table of Contents** tab (first tab in the top menu bar of the bottom sheet)

### Visual Design

#### Layout Structure
- Appears as a **full-height bottom sheet** or **half-sheet** overlay
- Slides up from the bottom with spring animation
- Contains a **grab handle bar** at top for dismissal

#### Chapter List Style
- **Vertical scrolling list** of chapters
- Each chapter row contains:
  - Chapter number/title (left-aligned)
  - Page number or location (right-aligned, gray text)
- Text uses **SF Pro** in the body size
- Chapter titles are in **regular weight**, dark text

#### Current Chapter Highlighting
- The currently active chapter is **bolded** or has a **blue tint**
- A small **dot indicator** or **blue bar** on the left edge marks the current position
- The list **auto-scrolls** to center the current chapter when opened

#### Additional TOC Features
- **Nested sections**: Sub-chapters are indented under parent chapters
- **Collapse/expand**: Chapters with sub-sections can be expanded
- **Progress indicators**: Small percentage or page count shown per section
- **Bookmark tabs**: Toggle between "Contents" and "Bookmarks" tabs at the top

#### Visual Details
- Background: Matches current theme (white/sepia/dark)
- Separator lines: 1pt light gray lines between rows
- Row height: Approximately 44–52pt (standard iOS list row)
- Active chapter: Blue accent color (#007AFF) for indicator
- Scroll indicators: Standard iOS vertical scroll indicator

## 4. Brightness Control

### Access Path
Tap menu button → **Theme & Settings** (brightness slider is in this panel)

### Implementation
- **Horizontal slider** at the top of the Theme & Settings panel
- Controls **screen brightness** (not just in-app opacity)
- Left icon: Small sun ☀️ (dimmer)
- Right icon: Large sun ☀️ (brighter)

### Visual Design
- Slider track: Light gray (#D1D1D6) with blue fill (#007AFF) for current value
- Thumb: White circle, approximately 28pt diameter
- Icons flanking the slider are SF Symbols:
  - `sun.min.fill` (small, low brightness)
  - `sun.max.fill` (large, high brightness)
- The slider spans nearly the full width of the panel (with padding)

### Behavior
- Real-time adjustment as slider is dragged
- Persists across reading sessions
- Independent of system brightness control

## 5. Search in Reader

### Access Path
Tap menu button → **Search** (magnifying glass icon in the top menu bar)

### Visual Appearance

#### Search Entry Mode
- **Top search bar** appears at the top of the reader view
- Standard iOS search bar design:
  - Rounded rectangle with gray background (#E5E5EA)
  - Magnifying glass icon on left
  - Placeholder text: "Search"
  - Clear button (X) on right when text is entered
- Keyboard slides up from bottom (standard iOS keyboard)

#### Search Results Display
- Results appear in a **list below the search bar**
- Each result row shows:
  - **Matching text snippet** with search term **highlighted in yellow**
  - **Page number** or chapter reference (right-aligned)
  - Results are ranked by relevance
- Tapping a result **jumps to that location** in the book
- Results are scrollable vertically

#### Navigation
- **Previous/Next** arrows to jump between matches
- Match counter: "1 of 23" displayed near the search bar
- **Cancel** button dismisses search and returns to reading position

### Animation
- Search bar slides down from top with animation
- Results list fades in below
- Dismissal: slide up + fade out

## 6. Bookmark Toggle

### Access Path
- **Quick access**: Tap menu button → bookmark icon
- **Long-press**: Some quick actions may include bookmark

### Icon Design

| State | Icon | Visual |
|-------|------|--------|
| **Not Bookmarked** | `bookmark` (outline) | SF Symbol: `bookmark` — outline only, gray/black stroke |
| **Bookmarked** | `bookmark.fill` (filled) | SF Symbol: `bookmark.fill` — solid filled, typically **orange/red** (#FF3B30) or **blue** (#007AFF) |

### Visual Details
- Icon size: Approximately 24–28pt
- State change: Instant toggle with subtle scale animation (spring bounce)
- When bookmarked, the icon fills with accent color
- The bookmark icon appears in the **bottom toolbar** of the settings panel
- Also accessible via the share sheet / more menu

### Additional Bookmark Features
- Bookmarks sync across devices via iCloud
- Bookmarked pages show a small colored tab on the page edge
- Bookmarks list accessible from the TOC panel's "Bookmarks" tab

## 7. Notes Panel

### Access Path
Tap menu button → **Notes** tab in the top menu bar of the bottom sheet

### Display Format

#### Notes List View
- Vertical scrolling list of all notes/highlights
- Each note row contains:
  - **Highlighted text** (shown in the highlight color)
  - **Note text** below the highlight (if user added a note)
  - **Chapter/location** reference (gray, smaller text)
  - **Date created** (optional, smaller text)

#### Highlight Colors Available
| Color | Hex | Use |
|-------|-----|-----|
| Yellow | `#FFEB3B` | Default highlight |
| Green | `#4CAF50` | Alternative |
| Blue | `#2196F3` | Alternative |
| Pink | `#E91E63` | Alternative |
| Purple | `#9C27B0` | Alternative |
| Orange | `#FF9800` | Alternative |

#### Visual Details
- Background: Matches current theme
- Highlighted text shown with colored background in the list
- Note text appears in regular body font
- Swipe actions: Swipe left to delete, swipe right to navigate to highlight
- Tap a note jumps to that location in the book

#### Adding Notes
- Select text → contextual menu appears
- Menu options: Copy, Highlight, Note, Share
- Tapping "Note" opens a text input field
- Notes are saved inline with the highlight

## 8. Bottom Sheet Behavior

### Container Design

#### Physical Characteristics
- **Grab handle bar**: 5pt tall, 36pt wide, rounded (capsule shape), centered at top
  - Color: Light gray (#C7C7CC) in light mode, dark gray (#636366) in dark mode
- **Background**: Matches current theme with slight blur/vibrancy
- **Corner radius**: 12pt at top corners (standard iOS sheet style)
- **Shadow**: Subtle drop shadow on presentation

#### Animation Behavior
- **Presentation**: Slides up from bottom with spring animation
  - Duration: ~0.35s with slight overshoot (spring damping ~0.8)
  - Initial position: Off-screen bottom
  - Final position: Approximately 50–70% of screen height
- **Dismissal**: 
  - **Swipe down**: Fling gesture dismisses the sheet
  - **Tap outside**: Tapping the dimmed area above dismisses
  - **Handle drag**: Dragging the handle down dismisses
  - Animation: Slides down with matching spring timing

#### Sheet Heights
- **Half sheet** (default): ~50% of screen height for Theme & Settings
- **Three-quarter sheet**: ~75% for Table of Contents / Notes
- **Full sheet**: ~90% for search results or detailed views

#### Interactive Behavior
- **Rubber banding**: Dragging past the top limit shows resistance
- **Velocity-based**: Fast fling dismisses even if not at threshold
- **Content scrolling**: When sheet content is scrollable, native scroll takes priority at certain drag thresholds

## 9. "More" Menu Options

### Access Path
Tap the **menu button** (two lines + three dots icon) in the bottom-right corner

### Menu Items

| Icon | Option | Description |
|------|--------|-------------|
| 📋 | **Copy** | Copy current page/selection text |
| 🔗 | **Share** | Open iOS share sheet for the book |
| 📖 | **Search** | Open search interface |
| 🔖 | **Bookmark** | Toggle bookmark on current page |
| 📝 | **Add Note** | Add a note to current selection |
| 🔒 | **Lock Orientation** | Lock to portrait/landscape |
| 📊 | **Reading Settings** | Open Theme & Settings panel |
| 📑 | **Table of Contents** | Open TOC panel |
| 💡 | **Brightness** | Quick brightness adjustment |
| 🔄 | **Share** | System share sheet |
| ⚙️ | **More** | Additional options |

### Context Menu (Long-Press on Text)
When text is selected, a contextual menu appears with:
- **Copy**
- **Highlight** (with color picker)
- **Add Note**
- **Share**
- **Dictionary/Look Up**
- **Translate**

### Visual Style
- Menu items are **text-only** or **icon + text** pairs
- Standard iOS action sheet styling
- Destructive actions (if any) shown in red
- Menu slides up from bottom
- Can be dismissed by tapping outside or swiping down

## 10. Additional Reader UI Elements

### Top Bar (Hidden by Default)
- **Page number / location**: Shown at top when tapped
- **Chapter title**: Brief display on tap
- Auto-hides after 2–3 seconds

### Progress Bar
- **Thin horizontal bar** at bottom of screen (above home indicator area)
- Shows overall book progress
- Draggable for quick navigation (scrubbing)
- Updates in real-time as you read

### Page Turn Options (iOS 16.4+)
- **Curl**: Classic skeuomorphic page curl animation
- **Slide**: Simple horizontal slide (default)
- **None**: Instant page change (no animation)

### Night Shift / True Tone Integration
- Works with iOS system-wide Night Shift
- True Tone adjusts color temperature based on ambient light
- Can be toggled in Theme & Settings

## Key Design Principles

1. **Minimal chrome**: Controls hidden by default, appear on tap
2. **Bottom-anchored**: All controls moved to bottom (iOS 16+) for reachability
3. **Sheet-based**: Everything uses iOS sheet/panel pattern
4. **Theme-adaptive**: All UI adapts to current background theme
5. **SF Symbols**: Consistent iconography using Apple's symbol set
6. **Spring animations**: All transitions use iOS spring physics
7. **Haptic feedback**: Subtle haptics on toggle/selection changes
