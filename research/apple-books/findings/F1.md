# F1 — Apple Books iPhone App: LIBRARY Screen UI Design Details

**Research Date:** 2026-07-15
**Scope:** iPhone app (iOS 18+ through iOS 26)
**Purpose:** Exact visual design details for replicating in a clone app

---

## 1. TOP-LEVEL APP STRUCTURE

### Tab Bar (4 tabs)
| Tab | SF Symbol | Notes |
|-----|-----------|-------|
| Reading Now | `book` | Open book with lines |
| Library | `books.vertical` | Stacked books |
| Book Store | `bag` | Shopping bag |
| Search | `magnifyingglass` | Standard search icon |

### Tab Bar Appearance (iOS 18-25)
- **Background:** Translucent with blur (`UIBlurEffect(style: .systemChromeMaterial)`)
- **Top separator:** 0.5pt hairline
- **Height:** ~49pt + safe area inset at bottom
- **Icons:** SF Symbols, ~25x25pt, outlined (unselected) / filled (selected)
- **Labels:** SF Pro, 11pt, regular weight
- **Unselected color:** `#1A1A1A` (light) / `#F5F5F5` (dark)
- **Selected color:** `#007AFF` (light) / `#0A84FF` (dark)

### Tab Bar (iOS 26+ Liquid Glass)
- **Shape:** Floating pill-shaped capsule, NOT full-width
- **Inset:** 21pt from left, right, and bottom edges
- **Background:** Liquid Glass translucent material
- **Search:** Separate circular Liquid Glass "island" to the right of tabs
- **Behavior:** Minimizes on scroll-down; expands on scroll-up

## 2. READING NOW TAB (Default Landing Screen)

The app opens to this tab by default. Shows books user is currently reading.

### Layout — Vertical scrolling with sections:

#### Section 1: "Continue Reading" (Top)
- Horizontal scrollable **carousel** of currently-reading book covers
- Cover size: **~120x180pt** with shadow
- **Book cover:** Portrait orientation, rounded corners ~3pt
- **Progress indicator** below cover (small progress bar or percentage)
- **Title** and **author** below cover
- This section appears only if user has books in progress

#### Section 2: "Want to Read"
- Same horizontal carousel format as Continue Reading
- Books user has added to their Want to Read list
- **"See All"** link on the right side of the section header to view the full list
- Tapping opens a detail view of all Want to Read books

#### Section 3: "Finished"
- Recently completed books
- Same carousel format

#### Section 4: "Samples"
- Book samples downloaded from the Book Store

### Section Header Style
- **Font:** SF Pro, large (~20-22pt)
- **Weight:** Bold
- **Color:** Primary label color (`#000000` light / `#FFFFFF` dark)
- **Alignment:** Left-aligned
- **"See All" link:** Right-aligned, accent blue, 15pt regular

## 3. LIBRARY TAB — Full Collection View

### View Modes (Toggle in top right)

#### Grid View (Default)
- **Columns:** **3 columns** in portrait, **4 columns** in landscape on iPhone
- Book covers displayed as grid tiles
- Cover aspect ratio: Portrait orientation (~2:3 ratio, standard book cover proportions)
- Cover size in grid: Calculated from column width (~105-110pt wide on standard iPhone)
- **Spacing:** ~8-12pt between covers horizontally and vertically
- **Margins:** ~16pt horizontal padding from screen edges
- **Book covers have:**
  - Rounded corners (~3pt radius)
  - Subtle **shadow:** Multi-layer, 2-4pt offset, 8-12pt blur (elevation effect)
  - Drop shadow color: Black at ~15-20% opacity
- **No title text under covers in grid view** — just the cover image
- Tapping a cover opens the book directly in the reader

#### List View
- Vertical list with:
  - **Book cover thumbnail** on left (~40x60pt, rounded corners ~3pt)
  - **Title** (bold, SF Pro ~17pt) on right of thumbnail
  - **Author** below title (regular weight, gray secondary text)
  - **Reading progress** or metadata on the far right
- Separator lines between rows: 0.5pt, system gray

### Sort Options (accessed via sort icon)
- **Recent** (default)
- **Title**
- **Author**
- **Manually** (drag to reorder)

### Collections / Sidebar Filtering
- Collections accessible via sidebar or dropdown:
  - **Want to Read** (built-in)
  - **Purchased** (all bought books)
  - **PDFs** (imported PDFs)
  - **Downloaded** (offline-available)
  - **Custom collections** (user-created)

## 4. BOOK CARD / COVER APPEARANCE (Detailed)

### Book Cover Image
- **Aspect ratio:** 2:3 (portrait), matching standard book cover proportions
- Cover images are the actual book cover artwork
- If no cover available: Default gray placeholder with book title text

### Cover Shadows (iOS Standard)
```
Shadow offset:    0pt horizontal, 2-4pt vertical (downward)
Shadow radius:    8-12pt blur
Shadow opacity:   15-20% (black)
Shadow color:     #000000
```
- Creates a subtle "lifted off the surface" effect
- More pronounced on Reading Now carousel covers (~120x180pt)
- Lighter on grid view covers (smaller size = less shadow needed)

### Rounded Corners
- Book covers: **~3pt** corner radius (very slight rounding)
- Library grid container has no additional rounding on tiles

### Cover Border
- **No visible border** around covers — shadow provides the separation from background

### Audiobook Covers
- May have a small headphone icon or "Audiobook" badge overlay
- Otherwise identical visual treatment

## 5. NAVIGATION BAR (Library Screens)

### iOS 18-25 Style
- **Large title** (34pt, bold, SF Pro) that transitions to small title (17pt, semibold) on scroll
- Background: Transparent with system blur (`systemChromeMaterial`)
- Translucent — content scrolls behind the bar
- Title text: `#000000` (light) / `#FFFFFF` (dark)

### Top-Right Actions
- **Edit** button (for multi-select, reorder)
- **View toggle** (grid/list icon)

### iOS 26+ (Liquid Glass)
- Adopts Liquid Glass material
- Floats above content
- Lensing effect bends light in real-time

---

## 6. SEARCH TAB

### Search Bar
- **Full-screen modal** sliding up from bottom
- Rounded-rectangle text field with magnifying glass icon (left), placeholder "Search", Cancel button (right, blue)
- Background: Blurred/translucent material matching current theme
- ~16pt horizontal padding

### Recent Searches
- Horizontal chip/pill tags below search bar
- Each tag: rounded-rectangle, gray background, gray text, "x" dismiss icon
- "Clear" text link on the right of section header "Recent Searches"

### Search Results
- Two categories with section headers:
  - **"Books"** — matches on title/author
  - **"Authors"** — matches on author name
- Each result row:
  - Book cover thumbnail (~40x60pt, rounded corners ~3pt)
  - Title (bold, single line, truncates with ellipsis)
  - Author name (regular weight, gray, single line)
- 0.5pt separator lines between rows
- Empty state: "No Results" centered, gray text

---

## 7. EMPTY STATE (Library Screen)

### When Library is Empty
- **Centered content** vertically and horizontally
- **Illustration/icon:** Subtle bookshelf or book icon (SF Symbol style)
- **Headline:** "Your Library is Empty" (SF Pro, ~20-22pt, semibold)
- **Subtext:** Instructional text explaining how to add books (SF Pro, ~15pt, regular, secondary gray)
- **Action button:** "Browse the Book Store" — blue accent color, tappable link
- Clean, minimal design following Apple's standard empty state pattern

## 9. TYPOGRAPHY

### Font System
- **All UI text:** SF Pro (system font)
- **Book content:** New York (default serif), or user-selected from: SF Pro, Athelas, Charter, Georgia, Iowan Old Style, Palatino, Songti, Zapfino

### Library Screen Typography Scale
| Element | Size | Weight |
|---------|------|--------|
| Large title (nav bar) | 34pt | Bold |
| Small title (nav bar, scrolled) | 17pt | Semibold |
| Section header ("Continue Reading") | ~20-22pt | Bold |
| Book title (list view) | 17pt | Semibold |
| Author name (list view) | 15pt | Regular |
| Tab bar labels | 11pt | Regular |
| "See All" link | 15pt | Regular (accent blue) |
| Sort/collection labels | 15pt | Medium |
| Caption/metadata | 12-13pt | Regular |

## 10. SPACING & LAYOUT

### Screen Margins
- **Horizontal:** ~16pt from screen edges (standard iOS inset)
- **Vertical:** Navigation bar occupies top; tab bar occupies bottom

### Grid View Spacing
- **Column gap:** ~8-12pt between book covers
- **Row gap:** ~8-12pt between rows
- **Grid columns:** 3 (portrait), 4 (landscape) on standard iPhone
- **Content inset:** ~16pt left/right, adjusted for safe area

### List View Spacing
- **Row height:** ~60-66pt (cover thumbnail height + padding)
- **Thumbnail to text gap:** ~12pt
- **Between rows:** 0.5pt separator line
- **Horizontal padding:** ~16pt

### Reading Now Carousel
- **Cover size:** ~120x180pt
- **Spacing between covers:** ~12-16pt
- **Horizontal scroll with snap** to nearest cover
- **Section vertical padding:** ~16-20pt between sections

---

## 11. ANIMATIONS & TRANSITIONS

### Tab Switch
- **Crossfade:** ~0.2s
- Standard iOS tab bar animation

### Modal Presentations
- **Spring animation:** ~0.35s, damping ~0.85
- **Modal present:** 0.4s spring from bottom
- **Modal dismiss:** 0.3s ease-out

### Book Open
- Tap cover → book opens in reader with slide-left transition
- Reader slide covers tab bar

### View Mode Switch (Grid ↔ List)
- Crossfade or morph animation between views
- Covers smoothly reflow into new layout

### Scroll Behavior
- Large title → small title on scroll (iOS standard)
- Content scrolls under translucent nav bar with fade edge effect

## 12. ICON STYLE (SF Symbols)

| Context | Size | Style |
|---------|------|-------|
| Tab bar icon | 25x25pt | Medium weight, regular |
| Nav bar button | 22x22pt | Regular weight |
| Grid/list toggle | 22x22pt | Regular weight |
| Sort icon | 22x22pt | Regular weight |
| Search icon | 22x22pt | Regular weight |

- **Unselected:** Outlined variant + gray
- **Selected/Filled:** Filled variant + accent color
- All icons use Apple's SF Symbols set (not custom icons)

---

## 13. COLLECTIONS SECTION

### Built-in Collections
- **Want to Read** — Books saved for later (added from store/book detail)
- **Purchased** — All books purchased
- **PDFs** — Imported PDF files
- **Downloaded** — Books available offline

### Want to Read Specifically
- Accessed via "Read Now" tab, "Want to Read" section near top
- **"See All"** link expands the section
- Each book shows cover + title + author + "three dots" menu
- **Three dots menu:** "Remove from Want to Read"
- **Edit mode:** Multi-select for batch removal (trash icon)

### User Collections
- Users can create custom collections
- Collections appear as filter options
- Each collection shows a cover stack thumbnail + title + book count

---

## 14. IMPORT / ADD BOOK

### From Book Store
- Tap "Book Store" tab
- Browse/search for books
- Tap "Want to Read" or purchase button
- Purchased books automatically appear in Library

### Import Files (PDFs/EPUBs)
- Via iOS Share Sheet from Files, Mail, etc.
- "Open in Books" option
- Imported files appear in Library under "PDFs" or general collection
- **No dedicated import button in Library screen** — relies on system share sheet

---

## 15. STATUS BAR HANDLING

- Standard iOS status bar (time, signal, battery)
- On notched devices: Dynamic Island or notch area respected
- Navigation bar sits below status bar
- Content extends behind status area when scrolling (translucent nav bar)

| Component | Material |
|-----------|----------|
