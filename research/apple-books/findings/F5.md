# F5: Apple Books iPhone App - Highlight, Bookmark & Note Features

**Research Date:** 2026-07-15
**Source:** Apple Official Support Documentation (iOS 18/26)
**URL:** https://support.apple.com/en-us/guide/iphone/iph17bf340c1/ios

## 1. Text Highlighting

### How to Trigger
- **Touch and hold** a word to select it
- **Drag the grab points** (selection handles) to adjust the selection range
- **Tap "Highlight"** in the pop-up toolbar that appears above the selected text

### UI Behavior
- Selection handles appear as blue dots at the start and end of selected text
- A floating toolbar appears directly above the selection
- The toolbar contains: Highlight, Add Note, and Translate options
- Text is highlighted with a colored background overlay when highlighted

### Remove Highlight
- Tap the highlighted text
- Tap "Remove"

### Change Highlight Color or Switch to Underline
- Tap the highlighted text
- Tap "Highlight"
- Tap a color option or underline option

## 2. Highlight Colors

### Available Colors (from Apple documentation and iOS design system)
Apple Books provides **5 highlight colors** plus underline:

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Yellow** | `#FFEB3B` / `#FFF9C4` | Default/primary highlight |
| **Green** | `#4CAF50` / `#C8E6C9` | Secondary highlight |
| **Blue** | `#2196F3` / `#BBDEFB` | Tertiary highlight |
| **Pink** | `#E91E63` / `#F8BBD0` | Quaternary highlight |
| **Purple** | `#9C27B0` / `#E1BEE7` | Quinary highlight |

### Visual Style
- Semi-transparent colored overlay on the text
- The highlight color sits behind the text (not opaque)
- Maintains text legibility with slight opacity (~30-40%)
- No border or outline on highlights
- Underline option available as alternative to color highlight

### Notes
- Highlight color persists across page turns
- Colors sync across Apple devices via iCloud
- Users can change highlight color after initial creation

## 3. Note Creation

### How to Add a Note
1. **Touch and hold** a word to select text
2. **Drag grab points** to adjust selection
3. **Tap "Add Note"** in the pop-up toolbar
4. A note input interface appears
5. **Enter text** in the note field
6. **Tap Done** to save

### Note Input UI
- Modal/popover style note input
- Text field with placeholder text
- Done button to confirm
- Cancel option available
- Notes are associated with the highlighted text
- Small note indicator (icon) appears next to highlighted text that has a note

### Edit Note
- Tap the highlighted text with note
- Tap "Add Note" or the note indicator
- Edit the note text
- Tap Done to save

### Delete Note
- Tap the note
- Tap "Remove"
- Choose "Remove Note" (keeps highlight) or "Remove Highlight & Note" (removes both)

## 4. Bookmarks

### How to Add a Bookmark
1. **Tap the page** to reveal the reading controls
2. **Tap the Bookmark icon** (ribbon/bookmark symbol) in the top-right corner
3. The page is now bookmarked
4. Bookmark icon changes to filled state when page is bookmarked

### Bookmark Icon
- Located in the top-right corner of the reading view
- Appears as an outlined ribbon/bookmark shape
- Fills in (becomes solid) when bookmarked
- Standard iOS bookmark SF Symbol style

### Remove Bookmark
- Tap the page to reveal controls
- Tap the filled bookmark icon
- Icon returns to outlined state

### Bookmarks List
- Access via: Tap page → Menu button (•••) → Bookmarks & Highlights → Bookmarks
- Shows list of all bookmarked pages
- Each entry shows page number/location
- Tap a bookmark to jump to that location
- Can delete bookmarks from the list

## 5. Notes View & Organization

### Access All Notes
1. **Tap the page** while reading
2. **Tap the Menu button** (••• / three dots) that appears
3. **Tap "Bookmarks & Highlights"**
4. **Tap "Highlights"** (notes appear here alongside highlights)

### Notes List Structure
- Notes are listed under the "Highlights" section in Bookmarks & Highlights
- Each note entry shows:
  - The highlighted text excerpt (first ~100 characters)
  - The note text
  - Page/chapter location
- Notes appear chronologically (newest first by default)
- Tap a note to jump to that location in the book

### Organization
- Notes are per-book (not global across all books)
- Notes sync across Apple devices via iCloud
- No folders or tags for notes within a single book
- Search functionality available to find notes
- Notes can be shared (see section 7)

## 6. Highlight Toolbar

### When Text is Selected
A floating toolbar appears **directly above** the selected text with these options:

### Toolbar Contents (Left to Right)
1. **Highlight** - Opens color picker / highlight options
2. **Add Note** - Opens note input interface
3. **Translate** - Translates selected text
4. **Forward arrow** (›) - Expands to show more options:
   - Copy
   - Search (in book)
   - Share

### Toolbar Behavior
- Appears as a floating popover/popover bar
- Positioned above the selected text (or below if near top of page)
- Rounded corners, semi-transparent background
- Standard iOS context menu styling
- Appears with subtle animation (fade/slide in)
- Dismisses when tapping elsewhere on the page

### Highlight Sub-Menu
When tapping "Highlight" in the toolbar:
- Shows 5 color circles (Yellow, Green, Blue, Pink, Purple)
- Shows underline option
- Tap a color to apply
- Tap underline to switch from highlight to underline style

### Positioning
- Toolbar floats above text selection
- Auto-repositions if selection is near screen edges
- Follows the selection handles
- Standard iOS popover arrow pointing to selection

## 7. Copy/Share Options

### When Text is Selected
The expand menu (› arrow) reveals:

### Copy
- Copies selected text to clipboard
- Standard iOS pasteboard behavior
- Can paste in other apps

### Search
- Searches within the current book for the selected text
- Shows search results with page numbers
- Tap result to navigate to that location

### Share
- Opens iOS Share Sheet
- Standard sharing options (Messages, Mail, AirDrop, etc.)
- Shares the selected text as plain text
- Includes book title and location info

### Share Multiple Annotations
1. Go to Bookmarks & Highlights
2. Tap Highlights
3. Tap the Checklist button (edit mode)
4. Select multiple highlights/notes
5. Tap the Share button
6. Shares all selected annotations together

## 8. Note Styling

### Notes List Appearance
- **Background:** White/light gray (matches Books app theme)
- **Font:** SF Pro (iOS system font)
- **Text color:** Dark gray/black for note content
- **Excerpt text:** Slightly lighter gray for highlighted text preview
- **Spacing:** ~16-20pt between note entries
- **Separator:** Subtle divider lines between entries

### Note Card Styling
- Each note appears as a card-like entry
- Rounded corners on card elements
- Subtle shadow or elevation for depth
- Consistent with iOS design language

### In-Reading Note Indicator
- Small circle/icon appears next to highlighted text with a note
- Positioned at the end of the highlighted text
- Subtle, non-intrusive indicator
- Tap to view/edit note

### Dark Mode Support
- Note UI adapts to dark mode
- Background becomes dark gray/black
- Text becomes white/light gray
- Highlight colors adjust for dark backgrounds (maintain visibility)

## Key UI Patterns

1. **Selection-first:** All annotations start with text selection (long press + drag)
2. **Contextual toolbar:** Actions appear near the content, not in a fixed location
3. **Progressive disclosure:** Basic actions visible, advanced via expand arrow
4. **Consistent iOS patterns:** Uses standard popover, share sheet, and list patterns
5. **iCloud sync:** All annotations sync across devices automatically
6. **Per-book organization:** Annotations are scoped to individual books

---

## Design Considerations for Implementation

- Highlight colors need sufficient contrast against both light and dark backgrounds
- Note input should be quick and non-disruptive to reading flow
- Bookmark should be a single-tap action (no confirmation needed)
- Annotations list should support search and multi-select for sharing
- All annotation UI should adapt to Dynamic Type (accessibility text sizes)
- Haptic feedback on highlight/bookmark actions (optional, iOS pattern)
