# Research Brief: Apple Books Features

**Question:** What are all the visual design details, UI patterns, interaction behaviors, and feature set of Apple Books on iOS (iPhone) that a clone app should replicate?

**Scope:** Focus on the iPhone app (not Mac, iPad, or web). Cover: library screen, reading screen, all control panels, typography settings, theme system, bookmark/highlight/note features, search, gestures, animations, and any subtle design details (spacing, shadows, blur effects, haptics).

**Depth:** standard

**Date:** 2026-07-15

## Angles

1. **Library Screen** — layout, book grid, collections, recently read, book cards, spacing, animations
2. **Reader Screen** — page layout, typography, margins, page turn animation, gestures
3. **Reader Controls** — bottom toolbar, font/typography panel, theme panel, TOC panel, brightness
4. **Themes & Appearance** — Light/Sepia/Dark/Night exact colors, blur effects, translucency
5. **Bookmarks, Highlights & Notes** — how they work, UI for creating/managing them
6. **Search & Navigation** — search UI, TOC, bookmarks list, page navigation
